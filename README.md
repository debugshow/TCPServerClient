# TCPServerClient
TCP Server and Client UDP DES and RSA Sigain
1.	基本要求实现情况
基本要求 

–服务器与客户端两个进程间进行TCP通信
服务器开启监听服务
服务端存储客户端信息

–消息加密 DES
简要描述
首先建立TCP链接，开启监听服务，等待连接
客户端用TCP连接服务器，发送自己信息，获取客户端的信息
客户端同时启动UDP服务，进行通信
